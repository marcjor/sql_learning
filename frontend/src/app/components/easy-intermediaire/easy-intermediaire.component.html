La notion de table


On représente les données sous forme de tables. Une base de données va donc contenir plusieurs tables. Elle peut n'en contenir qu'une, bien sûr, mais c'est rarement le cas.

Une table représente un type de données organisées, dans notre exemple, des personnes.

Chaque table est organisée en colonnes, qui sont les caractéristiques de l'objet représenté par la table.

On a donc ici une colonne "nom", une colonne "prenom", une colonne "langue", une colonne "pays" et une colonne "id" qui nous permettent d'identifier les personnes individuellement (les noms et prénoms ne suffisent pas toujours).

L'état de notre table

Dans une table, il est possible de sélectionner, modifier et supprimer des données. Ceci en utilisant des mots et des syntaxes spécifiques.

Nous allons d'abord SELECTIONNER des données dans notre table.
